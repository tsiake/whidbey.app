## server-side routes
******
```
These are the server-side routes that will contain a server-side response.
apis are served through apiRouter.js
user accounts get verified through confLinkRouter.js
downloads go through downloadsRouter.js - (receipts, pdfs, etc)
the initial webpage gets sent through homeRouter.js, react-router handles the other page links
recaptcha is handled via recaptchaRouter.js
```
